git init: Инициализирует новый репозиторий Git в текущем каталоге.
git add <файлы>: Добавляет изменения в указанные файлы в индекс (stage) для последующего коммита.
git commit -m "Сообщение коммита": Фиксирует изменения в индексе с сообщением коммита.
git push: Отправляет изменения в удаленный репозиторий.
git status: Показывает состояние рабочего каталога и индекса.
